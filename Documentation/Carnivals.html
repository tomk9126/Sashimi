Carnivals

Managing Carnivals

Opening Carnivals

The carnival sidebar provides the list of open carnivals. Selecting the ‘+’ button and choosing ‘Open Carnival’ will allow you to select and open a carnival file. That file will be added to the carnival list.

Alternatively, you may use the Menu Bar: ‘File > Open > Carnival’

Saving Carnivals

Carnivals that have been modified but not yet saved will display an Asterix in their title. A warning will also appear if the user tries to close the carnival without saving their recent changes, as unsaved changes will be lost.

Carnivals may be saved by choosing ‘File > Save’ in the menu bar, or more simply, ⌘ + ’S’

Creating a New Carnival

In the carnival sidebar, choose ‘+’ then ‘New Carnival’ (Note: you may also use ‘File > New > Carnival’ in the menu bar)

The first screen lets you choose a name and date for your carnival. You can switch back and forth between the two screens using the tab bar.

The second screen lets you import your Athletes. Use the ‘Import from Database (.csv)’ to import athlete data from a Comma Separated Value type spreadsheet. For more info on how to do this, see [Link’]Importing Athletes’. You may also use the ‘+’ and ‘-‘ buttons to create and delete athletes.

To edit the athlete, simply select its row in the table and it will become editable.

Sort order may be adjusted by clicking the headings in the top row of the table. The heading in bold represents the current sort order, and the adjacent error icon indicates whether the sort is in ascending or descending order. To toggle this, click the heading again.

To find specific athletes, you may use the search and filter icons provided in the bottom left of the screen.

Once complete, select ‘Create’

Note: A carnival may be created with no athletes.

Editing a Carnival

Using the context menu on a carnival in the sidebar, and then clicking ‘Edit’ will allow you to edit the Name and Date of the sidebar.
