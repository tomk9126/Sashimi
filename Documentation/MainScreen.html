Main Screen

The Sidebar

The sidebar displays open carnivals. These may be deleted, edited, closed via the context menu, or by selecting ‘Carnival’ in the menu bar when a carnival is the focused window.

Clicking ‘+{icon}’, or by selecting ‘File > New > Carnival’ in the menu bar, the carnival creation menu will open. This will allow you to create a new carnival.

The sidebar may be collapsed using ‘{icon}’

Carnival Toolbar

The toolbar provides the title of the currently selected carnival, tab bar, and toolbar items.

The Tab Bar allows switching between Scoring and Athletes interfaces. The toolbar items of the top-right provide the same actions allowed via the context menu, and will change to reflect whether the Scoring or Athletes interface is shown. Event/Athlete-altering actions (Edit, Delete, etc.) will be greyed out if nothing is selected.

Data Table

The carnival toolbar allows the user to switch between Scoring and Athletes data tables.

The Scoring table displays the list of Events in the carnival. By selecting, or invoking the context menu, on an event will provide the same options as the toolbar.

The Athletes table provides the list of athletes in the Carnival. By selecting, or invoking the context menu, on an athlete will provide the same options as the toolbar.

These tables may be sorted by clicking on the header row. For example, to sort by Event name in alphabetical, click the header of the ‘Event name’ column. The heading in bold displays which column is the sort order, and the adjacent arrow will display whether the sort order is ascending or descending. This may be toggled by rechecking the heading.

Toolbar Actions

All the following actions may be invoked via the context menu (May either be Secondary-Click or ⌘ + Click) or by their respective submenus in the menu bar.

Scoring Actions:

  • {Icon} Scoring: Takes the user to the scoring menu for the selected event. (Only shown if the event hasn’t been scored)
  • {Icon} Rankings: Provides two options: 1. View Ranking Data, 2. Re-score event. (Only shown if the event has been scored already)
  • {Icon} Edit Event: Takes the user to the Edit Event screen, allowing Name, Gender, and Age to be re-configured (Not available when an event is scored)
  • {Icon} Delete Event: Deletes the event and its contents.
  • {Icon} New Event: Takes the user to the New Event Screen
    •

    Athlete Actions:

  • {Icon} Export all Athletes: Will export a .csv file with all athlete data, which may be reused for another carnival or used elsewhere.
  • {Icon} Edit Athlete: Allows the data entry to be edited (Will be dismissed when the athlete is no longer selected.
  • {Icon} Delete Athlete: Deletes the athlete. Note: Will not remove them from rankings, but will be unavailable if the event is re-scored.
  • {Icon} New Athlete: Adds a new athlete to the list, and allows it to be edited.
